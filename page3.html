Used inside a SELECT — exactly like any built-in function
SELECT
    customer_id,
    COUNT(*) AS total_orders,
    GetLoyaltyTier(COUNT(*)) AS tier
FROM zomato.orders
GROUP BY customer_id;
The function is called once per row — each customer gets their own tier computed inline.
